== Description ==

A shortcode that embeds a PDF file using `<embed>` html tag.

== Installation ==

1. Unzip and upload to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use syntax like `[pdf]http://example.com/sample.pdf[/pdf]` in your posts
